Types of display elements.

block elements: Consume Whole width Available

paragraphs, headings, divisions, forms, list etc.

In-Line elements:These get inserted in the same line as far as the space is available.Their width and height can not be defined except inline-block elements and block elements.

images, span, anchor etc.

format to define inline block element : { display: inline-block;}, if we set it to none , it will be removed from html.
