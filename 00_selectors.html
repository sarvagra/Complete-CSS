Selectors :

1-Simple Selectors:

i)Element selector

id is unique to one element.

ii)Class Selector

same class can be given to many elements

iii)Id selector

This selector selects through id.

2-Universal selectors

Use * to select all the content(universal) and it has highest precedency

3-Grouping Selector

we can select different ids, classes separated by commas
